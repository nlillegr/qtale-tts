# Q-Tale TTS — WordPress Plugin

[![Stable Tag](https://img.shields.io/badge/stable-2.6.25-blue)](readme.txt)
[![WP Tested](https://img.shields.io/badge/WP-6.8-green)](https://wordpress.org/)
[![License](https://img.shields.io/badge/license-GPLv2%2B-yellow)](LICENSE)

Embed Q-Tale text-to-speech players in WordPress posts and pages with the
`[qtale]` shortcode. Norwegian premium synthetic voice with 25+ languages,
multiple player designs, server-side audio caching.

## Features

- 🎙 **25+ languages** via Piper + Kokoro (local) + Azure Neural + Giellalt (Sami)
- 🎨 **Player Designer** — visual player customization in your Q-Tale dashboard
- 🌐 **3-tier per-device design** — separate config for Desktop / Tablet / Mobile
- 📦 **Cloud Storage offload** (v2.6.25+) — push older MP3 files to your own
  FTP, SFTP, Amazon S3 (EU regions), Wasabi, Cloudflare R2, or Backblaze B2
- 🤖 **Auto-injection** with smart text extraction (excludes navigation, etc.)
- 🌍 **Translation modal** (with Q-Translate addon)
- 🛒 **WooCommerce Subscriptions** integration

## Backend

Plugin talks to `api.qtale.no` via X-API-Key. Account signup at
[qtale.no](https://qtale.no/priser) — free trial available.

## Installation

1. Download the latest ZIP from
   [GitHub Releases](https://github.com/<TO-BE-CONFIGURED>/qtale-tts/releases)
   or your Q-Tale dashboard
2. WordPress Admin → Plugins → Add New → Upload Plugin → Activate
3. Settings → Q-Tale TTS → enter API key
